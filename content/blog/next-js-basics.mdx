# Next.js Basics

This is a simple post to test MDX rendering in Next.js.

## Key Features

- **Server Components**: Next.js 13+ uses React Server Components by default.
- **App Router**: File-based routing system.
- **API Routes**: Create API endpoints as part of your Next.js app.

## Code Example

```javascript
// app/page.js
export default function HomePage() {
  return (
    <main>
      <h1>Welcome to my website</h1>
      <p>This is a simple Next.js page</p>
    </main>
  );
}
```
